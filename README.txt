--------------------------------------------------------------------------------
-------------------------- OOHOO COURSE FORMAT CALENDAR ------------------------
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Description
--------------------------------------------------------------------------------

This course format displays a course in a calendar format.
Functionalities to display by week, month or year and hide/unhide weekends.

--------------------------------------------------------------------------------
Prerequisites
--------------------------------------------------------------------------------

None
Note: This new version only works from 2.4+ until 3.1

--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

1.	copy the folder renamed "calendar" into the moodle/course/format folder.
2.	Login to Moodle with admin rights. 
3.	In the Site Administration block, click on Notifications a new 
        notification for the installation of Calendar Format will shows
4.      Click on Install


Your done. Go into a course edit page and select Format => Calendar format

Note: You can improve the number of sections ("Number of weeks/topics" field)
from Settings 
        => Site administration 
            => Courses 
                => Course default settings
                    => Field "Maximum for number of weeks/topics"
